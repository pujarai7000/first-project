# SQL Product Analysis

## Project Overview

This project was created to apply and reinforce the SQL concepts learned through the **Introduction to SQL** and **Intermediate SQL** courses. 
Using a real product dataset, I practiced querying, filtering, sorting, grouping, aggregation, pattern matching, and data exploration techniques. 
The project demonstrates how SQL can be used to extract insights from product data and answer business-related questions.

## Dataset

The dataset contains product information including:

* Product Key
* Product Subcategory Key
* Product SKU
* Product Name
* Model Name
* Product Description
* Product Color
* Product Size
* Product Style
* Product Cost
* Product Price

The dataset consists of approximately **294 products** and was used to practice SQL analysis techniques on real-world style product data.

## Analysis Performed

### Product Exploration

* Counted the total number of products in the dataset.
* Identified unique product subcategories.
* Displayed product styles from the dataset.

### Product Color Analysis

* Counted the number of white products.
* Counted the number of black products.
* Identified black-colored products with prices greater than 1000.

### Product Price Analysis

* Identified the 5 cheapest products.
* Calculated the average product price.

### Product Style Analysis

* Found product styles that appear more than 5 times.

### Pattern Matching

* Identified product names that start with the letter "R".

## Business Questions Answered

1. How many products are in the dataset?
